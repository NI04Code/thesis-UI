Dynamic Kubernetes clusters often accumulate resource fragmentation and multi-dimensional resource skews, leading to stranded capacity and inefficient infrastructure utilization. While the Kubernetes Descheduler provides a mechanism for post-placement optimization, traditional policies optimize solely for compute symmetry using rigid thresholds. For modern microservice architectures, standard bin-packing can break network locality, separating dependent services across degraded network links and increasing application latency.

This thesis presents the implementation and evaluation of a multi-level descheduling architecture designed to resolve both multi-dimensional fragmentation and network performance degradation. First, we propose a custom \textit{ResourceDefragmentation} policy that utilizes a Resource Imbalance Index (RII) and Multi-Criteria Decision Analysis to geometrically align CPU and memory allocations, unlocking stranded capacity. Second, we introduce a dual-layer Network-Aware filtering mechanism that integrates real-time network telemetry to evaluate and block latency-degrading evictions.

The combined architecture was evaluated on a live multi-node deployment across load-balancing, consolidation, and multi-dimensional skew correction scenarios. The empirical results demonstrate that while the baseline \textit{ResourceDefragmentation} algorithm balanced the cluster's compute resources, it sacrificed application performance by evicting sensitive workloads, causing latency spikes of up to $\sim$167ms. In contrast, the proposed Latency-Aware filter successfully intercepted these network-degrading actions, maintaining the baseline network latency at $\sim$8ms. The evaluation also showed that this network protection did not compromise defragmentation efficiency; by steering the resource-defragmentation algorithm to target non-critical pods instead, the integrated architecture achieved an optimal geometric resource balance (reducing cluster imbalance to 17.9\% Mean Absolute RII) while protecting network integrity.
